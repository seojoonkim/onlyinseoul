## 🚀 사용법

### 바로 사용하기
`index.html`을 브라우저에서 열면 됩니다.

### 파일 구조
```
├── index.html    # 메인 HTML
├── style.css     # 스타일시트
├── db.js         # 📦 데이터베이스 (여기서 시술 정보 수정)
├── app.js        # 앱 로직
└── README.md
```

## ✏️ 데이터 수정/추가

`db.js` 파일에서 시술 정보를 수정할 수 있습니다:

```javascript
{
  name: "시술명",
  brand: "브랜드/제조사",
  mechanism: "작용 원리 설명",
  target: ["효과1", "효과2", "효과3"],
  duration: "효과 지속기간",
  downtime: "회복 기간",
  pain: 2.5,        // 0~5 (통증 수준)
  sessions: "권장 시술 횟수",
  price: "가격대",
  note: "참고사항"
}
```

## 🤝 기여하기

1. Fork this repository

3. Pull Request 보내기

### 기여 가이드라인
- 정확한 정보만 추가해주세요
- 가격은 대략적인 범위로 작성
- 새로운 장비/시술은 출처와 함께 PR 부탁드립니다

## ⚠️ 주의사항

- 가격/효과는 병원, 개인에 따라 다를 수 있습니다
- 의료 정보는 변경될 수 있으니 최신 정보를 확인하세요
